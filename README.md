# SCARA-Robot 
This is a 3 DOF Selective Compliance Articulated Robot Arm used to pickup 1 inch wooden cubes.

This repository includes Arduino source code and hardware configuration.

### Setup
Connect stepper drivers and vacuum gripper according to the pin table in the report.

Install the Stepper.h library, then upload file to Arduino.

Current motor speed is set to 10 RPM. Adjust motor.setSpeed() in code.
